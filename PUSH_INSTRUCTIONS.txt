========================================

1. إنشاء مستودع جديد:
   - اذهب إلى: https://github.com/new
   - اسم المستودع: gulf-unified-gateway

2. بعد إنشاء المستودع، قم بنسخ URL الخاص به واستخدمه في الأمر التالي:

========================================
3. قم بنسخ وتشغيل هذه الأوامر:
========================================

# إضافة الرابط البعيد
git remote add origin https://github.com/YOUR_USERNAME/REPOSITORY_NAME.git

# الدفع للمستودع
git push -u origin main

========================================
4. أدخل بياناتك:
   - Username: اسم المستخدم الخاص بك على GitHub
   - Password: كلمة المرور (أو Personal Access Token)
