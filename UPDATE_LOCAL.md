# تحديث المشروع محلياً

## المشكلة: التحديثات على GitHub وليست على جهازك

## الحل - أمر واحد فقط:

```bash
cd gulf-unified-gateway
git pull origin main
```

## أو استخدم GitHub Desktop:
1. افتح GitHub Desktop
2. اختر المستودع
3. انقر "Pull origin" أو "Fetch origin"

## أو نزّل ZIP جديد:
1. اذهب لـ: https://github.com/aamdjuqp-arch/gtam
2. انقر "Code" → "Download ZIP"
3. فك الضغط
